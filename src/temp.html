Три виджета, минимально стоит в каждом использовать по одной технике 1. js-анимация - нужн овычислить высоту дочернего элемента, а затем менять высоту родительского до 0 (лии обратно) Фактчиески, css нужен только для прописывания начальных классов 2. css-анимация и переходы- для второй задачи Фактически, тут два элемента для преобразования - кнопка и форма 2.1. Кнопка увеличивается при наведении, а при клике просто исчезает - переход 2.2. Форм при появлении ме ньшего размера и полностью прозрачная, сперва увелдичвается прозрачность за полсекунды до половины, за тем еще за полсекунды формавырастает в размере и становистя совсем прозрачной, и обратно - css внимация Возможно, стоит слушать окончание анимации и запускать какорй-то доп код 3. js-анимация Тут при каждом клике запускаем функцию, оберку с реквестанимэйшенфрэйм и замыканием на старт анимаци, при достижени верха, нужно хапустить вторую вункцию - удаления лайка из ДОМа Данные для анимации: Изображение - heart.png Высота, на которой исчезают лайки - 200px Макс. отклонение влево/вправо - 50px Скорость анимации - 500ms Функция анимации - ease-out (можете использовать другую) Возможные траектории (траектория выбирается рандомно): 0% - центр, 25% - влево, 50% - центр, 75% - вправо, 100% - центр 0% - центр, 25% - центр, 50% - вправо, 75% - влево, 100% - центр 0% - центр, 25% - центр, 50% - влево, 75% - вправо, 100% - центр 0% - центр, 25% - вправо, 50% - центр, 75% - влево, 100% - центр Важно: обратите внимание, что элементы после "исчезания" должны удаляться из DOM. Используйте для этого соответствующие события, а не таймауты.
